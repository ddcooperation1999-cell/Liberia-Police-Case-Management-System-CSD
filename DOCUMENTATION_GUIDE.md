## FAQ

**Q: How do I log in?**  
A: Go to http://localhost:3000 and use dortusnimely / dortusnimely

**Q: Where are all the features?**  
A: Read [FEATURE_LOCATION_MAP.md](FEATURE_LOCATION_MAP.md) for the feature directory

**Q: How do I use each feature?**  
A: See [ADMIN_FEATURES_GUIDE.md](ADMIN_FEATURES_GUIDE.md) for complete documentation

**Q: Can I import users in bulk?**  
A: Yes! Go to User Management tab and click "Bulk Create (Up to 10,000)"

**Q: How do I create a case?**  
A: Go to Case Management tab and click "Create New Case"

**Q: Can I print clearance certificates?**  
A: Yes! Go to Police Clearance Check tab, find a suspect, and click "Print"

**Q: What if servers crash?**  
A: Run the server restart commands above

**Q: Is this production-ready?**  
A: Yes! See [DEPLOYMENT.md](DEPLOYMENT.md) for deployment instructions

**Q: Where is my data stored?**  
A: In a local SQLite database at backend/police_cases.db

**Q: Can I access from another computer?**  
A: Yes, use the network IP instead of localhost (see [DEPLOYMENT.md](DEPLOYMENT.md))
